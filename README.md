# TO-DO-LIST-APP
This is a simple To-Do List application built using Python. It helps users manage their daily tasks efficiently by allowing them to add, view, and delete tasks. The project uses file handling to store tasks permanently so that data is not lost after closing the program.

🚀 Features
➕ Add new tasks
📋 View all tasks
❌ Delete completed tasks
💾 Save tasks permanently using file handling
🖥️ Simple and beginner-friendly command-line interface

🛠️ Technologies Used
Python
File Handling (TXT/CSV)

🎯 Purpose of Project
The main goal of this project is to practice Python programming concepts such as file handling, loops, and functions while building a real-world utility application.
